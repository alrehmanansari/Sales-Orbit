# Sales Orbit Design System v3

## Brand Colors
- Orbit Blue:   #4796E3  (dark: #6BB5FF)
- Orbit Purple: #9177C7  (dark: #B49EE8)
- Orbit Pink:   #CA6673  (dark: #E8909B)
- Spectrum Gradient: linear-gradient(90deg, #4796E3, #9177C7, #CA6673)

## Typography
- Font: Google Sans, DM Sans (body)
- Display: Google Sans Display (headings)
- Mono: JetBrains Mono

## Taglines
- Primary: "Sell Faster. Orbit Further."
- Pipeline: "Your pipeline. Always in motion."
- Data: "Every deal. Every rep. Every second."

## Key Utilities
- .glass — glassmorphism card
- .glass-strong — stronger glass
- .soft-3d — soft 3D card with gradient + shadow
- .aurora-bg — aurora gradient background with orbs
- .glow-blue / .glow-purple / .glow-pink — dark mode glows
- .glow-spectrum — full spectrum glow

## Design Principles
1. Glassmorphism on sidebar, modals, auth cards
2. Aurora backgrounds on key landing areas
3. Animated pill for active nav/tab states
4. Glow effects in dark mode only
5. Gradient accent bar on active elements
6. Soft-3D on stat cards
